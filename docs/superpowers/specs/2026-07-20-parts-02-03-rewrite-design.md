## 1. 重写原因与成功标准

现稿已经建立课程注册、导航、测试和发布骨架，但多处仍是讲义提纲：核心定理被压缩为短段落，证明策略与严格证明没有分层，例题和习题不足以支撑标注学时，第二、三部还存在不动点内容重复。重写不是语言润色，而是依赖链、单元边界、证明叙事、例题体系和应用资产的整体重建。

完成后的单元必须允许普通高校数学类本科生在缺少教师持续讲解的情况下独立学习。所有核心定理给出完整证明；每个核心证明说明障碍、路线、逐步论证、假设使用位置、易错边界和可迁移方法；每个单元至少有 2 个完整例题、2 个即时检验和 5 道带折叠完整解答的分层习题。

## 2. 结构与迁移原则

1. 保留第 5–12 章的编号、主题和章级顺序。
2. 现有学习单元可以合并、拆分、改名或替换；旧 URL 通过迁移页或重定向指向新单元。
3. 单元按证明依赖和可独立学习边界划分，通常为 1.5–2.25 学时，不得超过 2.25 学时。
4. 两部先共享一张定理依赖图，再分别写作；一项理论只设一个正式归属。
5. 第二、三部核心学时暂定为 42 与 40，写作后按真实学习负担复核。全书总学时随总基线同步调整，不从后续部分预先机械回收。

## 3. 内容所有权与边界

- 第二部正式负责数列极限、实数完备性的数列后果、子列、Cauchy 准则、扩展实数中的 `limsup/liminf`、闭区间压缩映射定理、迭代收敛和误差界。
- 第三部正式负责完整的函数极限类型、连续性、闭区间上的序列紧致性与整体定理、介值定理、连续自映射的不动点存在性和二分法。
- 第三部不重复证明压缩映射定理；它只比较存在证书、区间误差证书和压缩收敛证书。
- 导数、微分中值定理、Taylor 公式、Newton 法、洛必达法则和收敛阶术语留到第四部。
- 有限极限运算法则不扩张为未定式代数；扩展实数只用于序、确界、趋于无穷和上／下极限。

## 4. 第二部蓝图

第二部由现有 16 单元、32 学时调整为约 21 单元、42 学时，其中理论约 34、应用约 8。

### 第5章 数列极限与量词结构（5单元，约10学时）

1. 数列、尾部与无限过程；
2. `epsilon-N` 定义及量词顺序；
3. 极限证明工坊：构造 `N(epsilon)`；
4. 唯一性、有界性与最终性质；
5. 趋于正负无穷、扩展实数与数值观察的边界。

### 第6章 极限运算与序关系（4单元，约8学时）

1. 和、差、数乘、乘积法则的完整证明；
2. 倒数与商：分母远离零的证明；
3. 保序性、夹逼定理及典型应用；
4. 误差分解、稳定估计与计算实验。

### 第7章 单调性、完备性与收敛准则（4单元，约8学时）

1. 单调有界收敛定理及其确界证明；
2. 递推数列：不变区间、单调性和候选极限；
3. 闭区间套定理及二分结构；
4. 完备性命题之间的依赖关系与证明方法选择。

### 第8章 子列、Cauchy准则与上／下极限（8单元，约16学时）

1. 子列、聚点与原数列行为；
2. Bolzano–Weierstrass 定理的完整证明；
3. Cauchy 准则的必要性与充分性；
4. 闭区间压缩映射定理；
5. 不动点迭代、几何误差界与停止条件；
6. Python 实现、慢收敛和失败迭代对照；
7. 尾部上／下确界与扩展实数中的 `limsup/liminf`；
8. 实现上／下极限的子列、有限收敛判别与无界例子。

完备性证明主线固定为

$$
\text{确界原理}
\Longrightarrow \text{单调有界收敛}
\Longrightarrow \text{区间套}
\Longrightarrow \text{Bolzano--Weierstrass}
\Longrightarrow \text{Cauchy准则}.
$$

`limsup/liminf` 在该主线完成后建立。压缩映射是应用闭环，不承担完备性基础角色。

## 5. 第三部蓝图

第三部由现有 13 单元、26 学时调整为 20 个核心单元、40 核心学时，其中第 9–12 章共 19 单元，另有 1 个必修证明桥单元；理论约 32、应用约 8。除此之外另设 1 个约 2 学时的选读附录单元。

### 第9章 函数极限与局部行为（8单元，约16学时）

1. 定义域、聚点、邻域与去心邻域；
2. 有限点有限极限的 `epsilon-delta` 定义；
3. `delta(epsilon)` 构造与非线性估计训练；
4. 左、右极限及双侧极限判别；
5. 有限点处趋于正负无穷；
6. 自变量趋于正负无穷时的有限／无穷极限；
7. 局部有界性、保号性、极限运算与复合；
8. Heine 序列刻画：等价证明、反例构造与适用边界。

有限点有限极限是证明训练主线。其余函数极限类型全部给出正式定义、判定关系和至少一个完整证明例，但不平均配置题量，不引入未定式运算。

### 第10章 连续函数与连续运算（4单元，约8学时）

1. 点连续、集合上连续及其极限／序列刻画；
2. 连续函数的代数运算与复合；
3. 单侧连续、间断点分类与连续延拓；
4. 常见初等函数连续性、使用边界与附录接口。

正文可以把常见初等函数的连续性列作明确的先备事实，不把它们写成“显然”。附录分层补足依据：

- 必修证明桥，约 2 学时并计入核心：多项式、有理函数、绝对值与复合函数；
- 选读证明桥，约 2 学时且不计入核心：三角函数连续性的恒等式与标准不等式证明；
- 指数与对数只列明引用性质，不在第三部从头构造。

### 第11章 闭区间上的整体性质（3单元，约6学时）

1. 闭区间的序列紧致性与收敛子列；
2. 连续函数的有界性和最值定理；
3. 一致连续性与 Heine–Cantor 定理。

证明采用收敛子列与反证法。开覆盖紧致性只作后续课程前瞻，不在第三部证明 Heine–Borel 或两种紧致性的等价关系。

### 第12章 零点、不动点与迭代求解（4单元，约8学时）

1. 零点定理与介值定理；
2. 区间上连续映射的不动点存在性，以及存在、唯一、收敛的区别；
3. 二分法的不变量、误差界、停止条件和 Python 实现；
4. 介值存在证书、二分区间证书与第二部压缩收敛证书的比较。

## 6. 统一单元模板

每个学习单元依次包含：

1. 单元信息、先备知识与依赖链接；
2. 学习目标与牵引问题；
3. 探索、猜想或失败尝试；
4. 定义—定理—证明的理论核心；
5. 至少 2 个完整例题；
6. 至少 2 个即时检验；
7. 必要的短实验或专门应用环节；
8. 至少 5 道分层习题及页面内折叠的完整解答；
9. 常见误区、结论边界和下一单元接口。

核心定理单元的习题必须包含证明题、反例或辨错题、条件变化题和跨单元迁移题。复杂证明题可以先提供分级提示，但必须同时保留完整解答。

## 7. 案例与计算线

两部采用“一个主案例加多个边界对照”的组织方式。主案例继续使用

$$
x_{n+1}=\frac{1}{2+x_n},
$$

连接不变区间、不动点、压缩迭代和几何误差。对照案例覆盖振荡迭代、`1/n` 慢收敛、非唯一零点、不连续异号、二分法和无界极限，避免一个案例被强行用于所有概念。

理论单元可以插入短 Python 实验；完整应用单元必须按“问题来源—数学转化—算法思想—误差与适用条件—伪代码—Python—结果解释”展开。二分法和压缩迭代等核心算法在 `src/` 中保留唯一实现，并为正常输入、边界条件和错误契约编写自动化测试。

## 8. 验收与实施边界

### 数学验收

- 证明没有跳步、循环依赖或未声明的前提；
- 自然数约定、量词顺序、指标范围和端点不变量一致；
- 扩展实数不被用于未定式代数；
- 序列紧致性、介值存在性、唯一性和迭代收敛性彼此区分。

### 教学验收

- 单元可独立自学，例题形成梯度；
- 最低题量和完整解答覆盖达标；
- 证明路线与严格证明分层呈现；
- Python 只支持猜想、反例、算法和解释，不替代理论。

### 课程与发布验收

- 单元、章、部学时与注册表一致，单元不超过 2.25 学时；
- 新旧 URL 迁移明确，稳定定理和习题 ID 可追踪；
- 代码测试、课程校验、Quarto 渲染、MathJax、锚点和站内链接全部通过。

实施时先冻结两部定理依赖图和新单元注册表，再依次重写第 5–8 章、第 9–12 章及附录。每章完成后立即进行数学、课程和渲染验收。本设计不授权顺带改写第一部；第一部按全书新基线在后续独立任务中复审。
